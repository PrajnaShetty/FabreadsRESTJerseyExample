 RESTJerseyExample
 
 A simple goodreads.com type of application to demonstrate REST with Java (JAX-RS) using Jersey
 
 How to run
 1) Start server (i used Tomcat) and deploy application on server.
 2) Type below URLs in browser
	http://localhost:8080/FabreadsRESTJerseyExample/getbookdetails/<book_id>
	eg. http://localhost:8080/FabreadsRESTJerseyExample/getbookdetails/4
